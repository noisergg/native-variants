## Overview

The `createNVA` function creates a themed instance of Native Variants with access to your colors and all Tailwind CSS design tokens.

```tsx
import { createNVA } from "native-variants";

const { styled, theme, colorScheme } = createNVA({
  theme: {
    colors: {
      default: { /* light colors */ },
      dark: { /* dark colors */ },
    },
  },
});
```

## Creating Your Theme

### Basic Setup

```tsx
const { styled, theme, colorScheme } = createNVA({
  theme: {
    colors: {
      default: {
        primary: "#3b82f6",
        secondary: "#8b5cf6",
        background: "#ffffff",
        foreground: "#0a0a0a",
        muted: "#f4f4f5",
        border: "#e4e4e7",
      },
      dark: {
        primary: "#60a5fa",
        secondary: "#a78bfa",
        background: "#0a0a0a",
        foreground: "#fafafa",
        muted: "#27272a",
        border: "#27272a",
      },
    },
  },
});
```

### Type-Safe Colors

TypeScript ensures that `dark` and `default` color schemes have exactly the same keys. If you add a color to one scheme but forget the other, TypeScript will show an error:

```tsx
// ‚ùå TypeScript Error: 'dark' is missing 'accent' key
const { styled, theme } = createNVA({
  theme: {
    colors: {
      default: {
        primary: "#3b82f6",
        accent: "#f97316", // Added here
      },
      dark: {
        primary: "#60a5fa",
        // Missing 'accent' - TypeScript will error!
      },
    },
  },
});
```

## Theme Object

The `theme` object returned by `createNVA` contains:

### Your Semantic Colors

```tsx
theme.colors.primary     // "#3b82f6"
theme.colors.background  // "#ffffff"
theme.colors.foreground  // "#0a0a0a"
```

### Tailwind Color Palette

Full Tailwind CSS color palette with all shades:

```tsx

theme.palette.slate500   // "#64748b"
theme.palette.slate900   // "#0f172a"

theme.palette.blue500    // "#3b82f6"

theme.palette.amber500   // "#f59e0b"
```

## Using Theme in Styled

The `styled` function provides access to the theme via a factory function:

```tsx
const cardVariants = styled((ctx, t) => ctx({
  slots: ["root", "title", "content"],
  base: {
    root: {
      backgroundColor: t.colors.card,
      borderRadius: t.radii.lg,
      padding: t.spacing["4"],
      ...t.shadows.md,
    },
    title: {
      fontSize: t.fontSizes.lg,
      fontWeight: t.fontWeights.semibold,
      color: t.colors.cardForeground,
      marginBottom: t.spacing["2"],
    },
    content: {
      fontSize: t.fontSizes.base,
      color: t.colors.mutedForeground,
      lineHeight: t.fontSizes.base * t.lineHeights.relaxed,
    },
  },
  variants: {
    elevated: {
      true: {
        root: { ...t.shadows.xl },
      },
      false: {
        root: { ...t.shadows.sm },
      },
    },
  },
  defaultVariants: {
    elevated: false,
  },
}));
```

## Using Theme Outside Styled

You can use the `theme` object directly in your components:

```tsx
import { theme } from "@/lib/nva";

function MyComponent() {
  return (
    <View style={{
      padding: theme.spacing["4"],
      backgroundColor: theme.colors.background,
      borderRadius: theme.radii.lg,
    }}>
      <Text style={{
        fontSize: theme.fontSizes.lg,
        color: theme.colors.foreground,
      }}>
        Hello World
      </Text>
    </View>
  );
}
```

<Note>
  The `theme` object uses colors from the `default` (light) scheme.
  For dark mode support, use the `ThemeProvider` and `useTheme` hook.
</Note>

## Extending Default Tokens

You can import and extend the default Tailwind tokens:

```tsx
import { 
  extendTheme,
  tailwindColors,
  tailwindSpacing,
} from "native-variants";

// Use individual tokens
const customSpacing = {
  ...tailwindSpacing,
  "18": 72,
  "22": 88,
};

// Or use extendTheme helper
const myTheme = extendTheme({
  colors: {
    brand: "#ff6b00",
  },
  spacing: {
    "18": 72,
  },
});
```

## Color Scheme Output

The `colorScheme` object contains both color schemes and should be passed to `ThemeProvider`:

```tsx
const { colorScheme } = createNVA({
  theme: {
    colors: {
      default: { primary: "#3b82f6" },
      dark: { primary: "#60a5fa" },
    },
  },
});

// colorScheme = {
//   default: { primary: "#3b82f6" },
//   dark: { primary: "#60a5fa" },
// }

// Pass to ThemeProvider for dark mode support
<ThemeProvider colors={colorScheme}>
  <App />
</ThemeProvider>
```
